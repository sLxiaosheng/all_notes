## 排序

```python
#原地排序，按指定的顺序排列数据，然后用排序后的数据代替原数据
#sort()
data = sort(data)
#复制排序，按指定的顺序排列数据，然后用排序后的数据生成一个副本（原数据还在）
#sorted
data2 = sorted(data)
```
